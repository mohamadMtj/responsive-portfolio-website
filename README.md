# 💼 Responsive Portfolio Website

A modern and fully responsive personal portfolio website built using HTML5, CSS3, and Vanilla JavaScript.

This project showcases a clean user interface with a fixed navigation bar, responsive sidebar menu, and modern layout suitable for developers, designers, and freelancers.

---

## ✨ Features

- 📱 Fully Responsive Design
- 📌 Sticky Navigation Bar
- 📂 Sidebar Navigation
- ⚡ Smooth User Experience
- 🎨 Clean and Modern UI
- 📅 Automatic Copyright Year
- 🚀 Lightweight and Fast
- 💻 Built with Vanilla JavaScript
- 🌐 Cross-Browser Compatible
